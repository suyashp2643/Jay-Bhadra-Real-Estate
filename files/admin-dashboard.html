🔑 Step 1 — Connect to GitHub
This is the one-time setup that makes leads save permanently and sync across all your devices. Takes 2 minutes.
How to get your GitHub token — follow these steps:
1
Go to github.com → click your profile picture (top right) → Settings
2
Scroll to the bottom of the left sidebar → click Developer settings
3
Click Personal access tokens → Tokens (classic) → Generate new token (classic)
4
Give it a name like "JB Dashboard". Under scopes, tick the repo checkbox. Set expiry to No expiration. Click Generate token.
5
Copy the token immediately — GitHub shows it only once. It starts with ghp_
📋 Step 2 — What happens after setup
🌐
Client fills your website form
Their name, phone, BHK choice, and message are captured instantly.
📁
Lead saved to GitHub automatically
Goes into files/leads-data.json in your repo within seconds — permanent, never lost.
📊
Dashboard loads leads from GitHub
Open this dashboard on any phone, laptop, or browser — it always shows all leads with AI scores.
🔥
AI scores each lead automatically
Hot / Warm / Cold is calculated instantly. Hot leads show with a red badge in the sidebar so you never miss one.
📂 Lead data management
Export or clear your lead data.
Leads are saved to GitHub (permanent) and to this browser (backup). Clearing here removes the browser copy only — GitHub copy stays safe. To delete from GitHub, edit files/leads-data.json directly in your repo.
